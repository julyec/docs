dio_read

(PHP 4 >= 4.2.0, PHP 5 < 5.1.0)

dio_read — 从文件描述符读取字节

说明

dio_read(resource $fd, int $len = 1024): string

dio_read() 函数读取并返回描述符 fd 指定文件中的 len 字节。

参数

fd

由 dio_open() 返回的文件描述符。

len

要读取的字节数。如果未指定，dio_read() 读取 1K 大小的块。

返回值

从 fd 读取的字节。

参见

  • dio_write() - 截取可选长度的数据写入文件
