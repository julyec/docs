xml_error_string

(PHP 4, PHP 5, PHP 7, PHP 8)

xml_error_string — 获取 XML 解析器的错误字符串

说明

xml_error_string(int $error_code): ?string

获取与指定 error_code 关联的 XML 解析器错误字符串。

参数

error_code

由 xml_get_error_code() 返回的错误代码。

返回值

返回带有错误 error_code 文本描述的字符串，若没有与之对应的描述，则返回 null。

<?php
$p = xml_parser_create();
xml_parse_into_struct($p, $data, $vals, $index);
//This call prints error
echo 'ERROR if any '.xml_error_string(xml_get_error_code($p));
xml_parser_free($p);
echo "Index array\\n";
print_r($index);
echo "\\nVals array\\n";
print_r($vals);
//No Error left here to be printed
echo 'ERROR if any '.xml_error_string(xml_get_error_code($p));
exit;
