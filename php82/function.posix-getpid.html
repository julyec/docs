posix_getpid

(PHP 4, PHP 5, PHP 7, PHP 8)

posix_getpid — 返回当前进程 id

说明

posix_getpid(): int

Return the process identifier of the current process. 返回当前进程的 id

参数

此函数没有参数。

返回值

返回进程 id 号，是整型（int）。

示例

Example #1 posix_getpid() 的使用例子

<?php
echo posix_getpid(); //8805
?>
