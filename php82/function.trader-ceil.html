trader_ceil

(PECL trader >= 0.2.0)

trader_ceil — Vector Ceil

说明

trader_ceil(array $real): array

Calculates the next highest integer for each value in real and returns the resulting array.

参数

real

浮点数数组。

返回值

Returns an array with calculated data or false on failure.
