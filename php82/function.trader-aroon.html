trader_aroon

(PECL trader >= 0.2.0)

trader_aroon — Aroon

说明

trader_aroon(array $high, array $low, int $timePeriod = ?): array

参数

high

高价，浮点数数组。

low

低价，浮点数数组。

timePeriod

Number of period. Valid range from 2 to 100000.

返回值

Returns an array with calculated data or false on failure.
