trader_ad

(PECL trader >= 0.2.0)

trader_ad — Chaikin A/D Line

说明

trader_ad(
    array $high,
    array $low,
    array $close,
    array $volume
): array

参数

high

高价，浮点数数组。

low

低价，浮点数数组。

close

收盘价，浮点数数组。

volume

成交量，浮点数数组。

返回值

Returns an array with calculated data or false on failure.
