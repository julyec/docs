ReflectionFunctionAbstract::getName

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

ReflectionFunctionAbstract::getName — 获取函数名称

说明

public ReflectionFunctionAbstract::getName(): string

获取函数名称。

参数

此函数没有参数。

返回值

返回函数名称。
