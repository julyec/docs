getrandmax

(PHP 4, PHP 5, PHP 7, PHP 8)

getrandmax — 显示随机数最大的可能值

说明

getrandmax(): int

返回调用 rand() 可能返回的最大值。

参数

此函数没有参数。

返回值

rand() 返回随机数可能返回的最大值

参见

  • rand() - 产生一个随机整数
  • srand() - 播下随机数发生器种子
  • mt_getrandmax() - 显示随机数的最大可能值
