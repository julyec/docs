acosh

(PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

acosh — 反双曲余弦

说明

acosh(float $num): float

返回 num 的反双曲余弦值，即其双曲余弦为 num 的那个值。

参数

num

要处理的值

返回值

num 反双曲余弦

参见

  • cosh() - 双曲余弦
  • acos() - 反余弦
  • asinh() - 反双曲正弦
  • atanh() - 反双曲正切
