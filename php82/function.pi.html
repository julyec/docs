pi

(PHP 4, PHP 5, PHP 7, PHP 8)

pi — 得到圆周率值

说明

pi(): float

返回圆周率的近似值。也可以使用 M_PI 常量，该常量产生与 pi() 完全相同的结果。

参数

此函数没有参数。

返回值

圆周率（pi）的浮点近似值。

示例

Example #1 pi() 例子

<?php
echo pi(); // 3.1415926535898
echo M_PI; // 3.1415926535898
?>
