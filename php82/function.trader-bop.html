trader_bop

(PECL trader >= 0.2.0)

trader_bop — Balance Of Power

说明

trader_bop(
    array $open,
    array $high,
    array $low,
    array $close
): array

参数

open

开盘价，浮点数数组。

high

高价，浮点数数组。

low

低价，浮点数数组。

close

收盘价，浮点数数组。

返回值

Returns an array with calculated data or false on failure.
