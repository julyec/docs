abs

(PHP 4, PHP 5, PHP 7, PHP 8)

abs — 绝对值

说明

abs(int|float $num): int|float

返回参数 num 的绝对值。

参数

num

要处理的数字值

返回值

num 的绝对值。 如果参数 num 是 float，则返回的类型也是 float，否则返回 int（因为 float 通常比 int 有更大的取值范围）。

更新日志

版本 说明
8.0.0 num 不再接受支持数字转换的内部对象。

示例

Example #1 abs() 例子

<?php
var_dump(abs(-4.2));
var_dump(abs(5));
var_dump(abs(-5));
?>

以上示例会输出：

float(4.2)
int(5)
int(5)
