SplStack 类

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

简介

SplStack 类的主要功能是通过将迭代模式设置为 SplDoublyLinkedList::IT_MODE_LIFO 来提供使用双向链表实现的栈。

类摘要

class SplStack extends SplDoublyLinkedList {

public const int SplDoublyLinkedList::IT_MODE_FIFO;
public const int SplDoublyLinkedList::IT_MODE_DELETE;
public const int SplDoublyLinkedList::IT_MODE_KEEP;

示例

Example #1 SplStack 示例

<?php
$q = new SplStack();
$q[] = 1;
$q[] = 2;
$q[] = 3;
foreach ($q as $elem) {
echo $elem."\n";
}
?>

以上示例会输出：

3
2
1
