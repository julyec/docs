ZipArchive::count

(PHP 7 >= 7.2.0, PHP 8, PECL zip >= 1.15.0)

ZipArchive::count — Counts the number of files in the archive

说明

public ZipArchive::count(): int

参数

此函数没有参数。

返回值

Returns the number of files in the archive.
